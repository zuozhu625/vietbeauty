### 1. DQA系统说明

所有文档现在都包含以下DQA关键信息：
- **服务类型**: 医院自动问答系统
- **医院数据**: 122家
- **问答类型**: 6种（资质、等级、服务、地址、联系、医生）
- **定时频率**: 每15分钟
- **分类标签**: "Tư vấn bệnh viện"（绿色主题）
- **API端点**: /api/dqa/*

### 2. 前端优化说明

前端文档中新增以下优化内容：
- **Hero高度**: 缩小35%（pt-32 pb-16 → pt-20 pb-10）

- **标签样式**: 添加whitespace-nowrap防止换行
- **医院问答**: 绿色主题标签，无icon

### 3. API接口文档

后端文档中新增DQA API章节：
- GET /api/dqa/status - 服务状态
- GET /api/dqa/stats - 运行统计
- GET /api/dqa/hospitals - 医院清单
- POST /api/dqa/generate - 生成问答
- POST /api/dqa/scheduler/:action - 控制任务
- GET /api/dqa/chain-hospitals/* - 连锁医院管理

### 4. 部署流程更新
